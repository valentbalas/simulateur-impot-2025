## Déploiement Streamlit Community Cloud (gratuit)
1. Uploadez les 5 fichiers sur GitHub (dépôt public)
2. Connectez-vous sur https://share.streamlit.io
3. "New app" → sélectionnez votre repo → Main file: app.py → Deploy

## Lancement local
```bash
pip install -r requirements.txt
streamlit run app.py
```
